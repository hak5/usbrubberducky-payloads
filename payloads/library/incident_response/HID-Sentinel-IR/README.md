# HID Sentinel IR v1.0

Lightweight defensive framework against BadUSB/HID attacks for Windows 10/11.

## Features

- Detects rogue USB/HID devices
- Monitors suspicious PowerShell activity
- Uses low-noise event monitoring
- Generates local incident logs
- Uses PASSIVE_WINDOWS_DETECT for safer execution

## Logs

C:\Windows\Temp\sentinel.log
