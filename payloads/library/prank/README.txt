# ShadowShutdown

Advanced Rubber Ducky payload designed for controlled cybersecurity research and red team lab environments.

This payload demonstrates:
- Automated execution of administrative commands
- Modification of selected Microsoft Defender protections
- Automated system shutdown procedures

---

## Features

- Automated command execution
- Multiple Defender-related configuration changes
- Fast payload deployment
- Automatic shutdown sequence
- Designed for USB HID attack simulations

---

## Usage

Deploy the payload to a compatible Hak5 Rubber Ducky device and execute it in an isolated, authorized testing environment.
