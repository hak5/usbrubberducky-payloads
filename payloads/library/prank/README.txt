# ShadowShutdown

Advanced Rubber Ducky payload designed for controlled cybersecurity research and red team lab environments. This payload demonstrates automated execution of administrative commands, modification of selected Microsoft Defender protections, and automated system shutdown procedures.

## Features

* Automated command execution
* Multiple Defender-related configuration changes
* Fast payload deployment
* Automatic shutdown sequence
* Designed for USB HID attack simulations

## Usage

Deploy the payload to a compatible Hak5 Rubber Ducky device and execute it in an isolated, authorized testing environment.

## Disclaimer

This project is intended strictly for educational purposes, security research, authorized penetration testing, and lab simulations only. Unauthorized use against systems you do not own or have explicit permission to test may violate laws and regulations.

## Requirements

* Hak5 Rubber Ducky
* Windows target system
* Authorized testing environment
